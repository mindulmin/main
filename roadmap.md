# 시장 빈틈 공략 10가지 (내부 로드맵)

이 문서는 사이트에 노출하지 않는 내부 로드맵 요약입니다.

1) 무료 학습 흐름이 끊기지 않는 경험  
근거: 광고/하트/에너지로 학습이 중단된다는 불만이 반복됨.  
반영: “1분 퀴즈/1문장 교정”은 로그인 없이 즉시 사용 + 광고는 본문 중간 1~2개로 최소화

2) ‘왜 틀렸는지’ 설명 + 재발 방지  
근거: 앱이 정답 중심/피드백 얕다는 불만 존재.  
반영: 교정 결과에 규칙 1줄 + 예문 2개 + 미니퀴즈 1개 자동 생성

3) 개인 오류 아카이브(나만의 오답노트)  
근거: 유저는 “내가 반복하는 실수”를 추적하고 싶어함.  
반영: 교정/퀴즈 결과를 오류 태그(조사/시제/어순)로 저장, 다음 퀴즈에 반영

4) 한국어 띄어쓰기/조사 교정에 특화  
근거: 한국어는 띄어쓰기·조사·활용이 구조적으로 어렵고 교정 수요 큼.  
반영: “Korean Writing Fix” 도구 + “왜 이 띄어쓰기가 맞는지” 초간단 설명

5) 발음 교정의 ‘구체적 처방’  
근거: AI 회화 앱이 발음 피드백이 기초적/관대하다는 지적.  
반영: 발음 진단 → 입모양/혀 위치 → 최소대립쌍 퀴즈 한 세트 제공

6) K-드라마/가사 기반 학습 도구  
근거: 학습 동기는 강하지만 도구가 파편적.  
반영: “대사 1줄 → 표현 설명 → 비슷한 표현 3개 → 쉐도잉” 템플릿

7) 업무/커리어 상황별 영어 교정  
근거: AI 번역 시대에도 업무 커뮤니케이션은 중요하다는 논지.  
반영: 이메일/면접/회의/메신저별 템플릿 + 톤/격식 레벨 교정

8) 다국어 UI + 다국어 학습언어 조합 확장  
근거: 다국어 접근성이 학습자 증가에 영향을 준 흐름이 있음.  
반영: 영어↔한국어뿐 아니라 “스페인어 UI로 한국어 배우기”처럼 조합 확장

9) 짧고 반복 가능한 ‘마이크로’ 학습  
근거: 모바일 습관에 맞는 짧은 학습이 계속 강화되는 흐름.  
반영: “오늘의 60초 퀴즈” + “누적 스트릭”을 웹에서 구현

10) 광고 수익형(애드센스) SEO 최적 도구형 페이지  
근거: 교육/온라인 학습이 니치로 자주 언급됨.  
반영: 블로그 + 도구 페이지 조합(퀴즈/교정/템플릿)으로 페이지뷰·체류시간 강화
